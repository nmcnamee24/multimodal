# The Platform Machine

Interactive model about AI-generated music, streaming platforms, recommendation systems, and the redistribution of value in the digital music economy.

## What It Is

The app explains how cheaper music production can increase platform power. Users move through a clickable flow, adjust an AI production-pressure scenario, compare value-capture models, and see why recommendation systems become more important when track supply expands faster than listener attention.

## Tech Stack

- React
- TypeScript
- Vite
- Lucide React icons

```bash
npm install
npm run dev
```
