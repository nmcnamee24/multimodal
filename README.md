# The Platform Machine

Interactive museum-style infographic for a multimodal remix project about AI-generated music, streaming platforms, recommendation systems, and the redistribution of value in the digital music economy.

## What It Is

The app presents a scholarly argument as an exhibit-wall experience. Visitors move through stations for supply growth, algorithmic gatekeeping, and value distribution using a mix of textual, visual, spatial, gestural, and optional aural modes.

## Tech Stack

- React
- TypeScript
- Vite
- Lucide React icons

## Run Locally

```bash
npm install
npm run dev
```

## Check Before Submitting

```bash
npm run lint
npm run build
```
